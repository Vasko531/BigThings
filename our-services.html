Remapping a car changes the manufacturer’s default settings and software on the ECU, replacing it with new software which can be tweaked and customized to the owner’s specifications (within legal limitations).
When a vehicle is remapped, the old ECU software is overwritten when the customized software is plugged in to the car’s  OBD port. This simple functionality has made the process of tuning a vehicle considerably easier, however it is not possible to be applied in all of the cases.
As mentioned above, remapping allows you to alter the performance of your vehicle by altering how the engine drives the car, however, it’s how the driver chooses to use the tuned car that ultimately decides what effects the tuning will have.


Remapping a vehicle can also have an immediate effect on speed and power; in some cases, remappers claim to have increased horsepower by 30-40BHP, and torque up to 80NM!
A car’s engine can also become more responsive, allowing the driver to overtake more easily. This can also cut down on the number of gear changes the driver has to make, allowing them to handle situations that require more torque without dropping down to lower gears. Again, this can help boost fuel economy if used strategically.
DPF / FAP OFF – Disabling diesel particulate filter making it even physically removable after which the engine continues to operate normally without any consequences; at the same time you will save a lot of money as filters are very expensive, and their replacement cannot guarantee that the error will not reappear after some time.
An increasing number of imported used vehicles have a built-in Diesel Particulate Filter or DPF, which, after switching to local operating conditions, starts causing headaches to their owners. DPF gets clogged due to several factors including the dirty fuel, distance travelled, driving style, etc. The main feature of a vehicle with clogged particulate filter is drastic power loss which largely affects the traffic safety. As cleaning is only a temporary solution and the replacement cost nears the amount of 1000 EUR we started offering a service of disabling DPF and FAP (permanent removal of filter).
The filter is physically removed from the exhaust system and the software in the ECU is adjusted so that it no longer notices differential pressure sensors at the entrance and exit point of the filter. In this manner it is avoided the possibility of a moving car going into “limp home” mode with a reduced power.

 EGR OFF – EGR is a valve that serves for exhaust gas recirculation when the car is idling and primarily serves as an aspect of environmental protection; in most cases it is not enough to simply mechanically block the EGR but it must be disabled in motor electronics as it detects the corrupt EGR and it often happens the engine runs in limp mode, or in the best case, light on the instrument panel signals error. We offer solutions that allow a permanent removal of EGR valve without any consequences to engine operation. 
DTC -Permanently removing fault codes from the engine management system is a very useful process when modifying a vehicle.
Removing components such as swirl flaps, EGR valves, customised modules for purpose of performance modification or Motorsport, will often cause a fault code to be stored in the ECU memory along with lights on the dash and 'Limp Home' mode.
By recoding the ECU data and permanently removing the specific fault code we can prevent the dash lights from being displayed and prevent the vehicle from entering in 'Limp Home' mode.
We endeavour to remove most of the DTC codes or also known as OBD codes from Engine Control Unit (ECU) in a process involving reading the ECU files, deleting the fault code from the ECU fault code system and then writing the new files back to the ECU. Please note that we do this using dedicated tools and software engineering; we don’t clean the fault code with a diagnostic tool. By following our approach, we guarantee that the error will not come back.
